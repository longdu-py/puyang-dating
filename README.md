# 濮阳同城交友相亲平台
濮阳本地单身男女相亲交友前端网站，纯静态页面，无需后端，支持GitHub Pages一键部署。
## 本地功能
1. 濮阳区县筛选：华龙区、濮阳县、清丰、南乐、范县、台前
2. 年龄、职业、择偶要求精准筛选同城单身
3. 用户注册/登录、个人资料、相册上传模拟
4. 心动喜欢、访客记录、收藏嘉宾
5. 一对一私信聊天（本地存储消息不丢失）
6. 濮阳线下相亲活动发布、报名功能
7. 每日同城匹配推荐、缘分速配
## 技术栈
HTML5 + Bootstrap5 + JavaScript localStorage
## 部署方式
1. 下载全部文件本地打开index.html预览
2. 上传至GitHub仓库，开启Pages即可线上访问
